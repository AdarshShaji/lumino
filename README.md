# Lumino - Next.js React App

This is a Next.js React application with Google authentication using Firebase.

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up your Firebase project and add the configuration to `.env.local`
4. Run the development server: `npm run dev`

## Features

- Google login and registration using Firebase
- Next.js React framework
- Environment variable configuration

## Environment Variables

Make sure to set up the following environment variables in your `.env.local` file:

- NEXT_PUBLIC_FIREBASE_API_KEY
- NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN
- NEXT_PUBLIC_FIREBASE_PROJECT_ID
- NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET
- NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID
- NEXT_PUBLIC_FIREBASE_APP_ID
